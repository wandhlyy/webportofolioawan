=== README - Web Portofolio ===

Nama Proyek: Web Portofolio
Deskripsi: Web Portofolio adalah platform untuk mengumpulkan dan menampilkan proyek-proyek kreator berbakat.

== Cara Instalasi ==

Clone repository ini ke lokal:

git clone <repository-url>

Masuk ke direktori proyek:

cd web-portofolio

Instal dependensi menggunakan Composer:

composer install

Duplikasi file .env.example menjadi .env dan atur konfigurasi database.

Generate application key:

php artisan key:generate

Jalankan migrasi database:

php artisan migrate

Jalankan server lokal:

php artisan serve

== Fitur Utama ==

Halaman Dashboard

Halaman Login & Registrasi

Halaman Admin (Hanya bisa diakses oleh admin1234@gmail.com)

Fitur Logout
